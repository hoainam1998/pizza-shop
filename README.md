<h1>Pizza shop store</h1>

## Description
Basic the product are helping to sell fast food, they include a few simple features like below:
  - Management product, category, product ingredient.
  - Create system user, mutating user information.
  - Allow user login to use system.
  - Authorized and authenticated by user role.
  - Report capital and revenue in real-time, or for a specific time.
  - Using a socket client to receive information that is emitted from the server. For some case like as user information updated or adding revenue data chart by time interval.
  - User logout.  
 
This is my personal project, whose intended purpose is to improve coding skills. Not for business.
The project is being separated into ui and api folder.  
Every detail of information is being presented in each folder. See below.
